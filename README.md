## 🛠️ Core Project Architecture
This repository contains a fully integrated, multi-page web prototype optimized for smooth cross-navigation. The ecosystem consists of the following workflows:

*   **`index.html` (Homepage):** The main hub featuring a dynamic product filter script, live hashtag generation, and a seamless continuous-loop product showcase carousel.
*   **`about.html` (About Us):** Overview of Pockverse's core mission, market vision, and digital production solutions.
*   **`services.html` (Products & Services):** Interactive packaging creator layout highlighting 3D CAD structure mapping and API configuration capabilities.
*   **`designer-marketplace.html`:** Connects commercial clients with structural packaging and branding design specialists.
*   **`cart.html` & `account.html`:** The checkout workflow and unified merchant/designer dashboard portals.

---

## ⚡ Features Implemented
*   **Seamless Relative Routing:** Completely unified global `<header>` navigation across all documents, allowing users to move between sections and pages without breaking URL paths.
*   **Dynamic Data Filtering:** JavaScript-driven interface on the homepage that filters products instantly by shape, industry, or utility without requiring a database.
*   **Infinite CSS Carousel:** Animation-driven, infinite scrolling carousel showcasing popular packaging categories.
*   **Fully Responsive Layouts:** Flexible CSS styling that adapts dynamically across desktop monitors, tablets, and mobile devices.

---

## 🚀 Technical Framework
*   **Frontend Language:** HTML5 / Semantic Web Architecture
*   **Styling Engine:** Native CSS3 (Flexbox, Grid Layouts, Keyframe Animations)
*   **Scripting Logic:** Vanilla JavaScript (ES6+ DOM manipulation and event filtering)
*   **UI Components:** Google Fonts (Roboto / Poppins) & FontAwesome V6 Icons
